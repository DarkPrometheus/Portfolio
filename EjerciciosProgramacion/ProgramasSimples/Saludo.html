Instrucciones

Escriba un programa que pida al usuario que escriba su nombre, y lo salude llamándolo por su nombre.

Resultado esperado

Ingrese su nombre: Perico Hola, Perico

Explicacion

Este ejercicio es un ejercicio básico que se asemeja al famoso "Hola Mundo" en el mundo de la programación, pero en lugar de simplemente imprimir "Hola, Mundo", estamos interactuando con el usuario.

Pasos clave

  • Solicitud de entrada: En este ejercicio, el primer paso es solicitar al usuario que ingrese su nombre. Esto se puede lograr utilizando una función de entrada en el lenguaje de programación que estés utilizando.
  • Almacenamiento en una variable: Una vez que el usuario ingrese su nombre, debes almacenarlo en una variable. En este caso, necesitarás una variable para almacenar el nombre del usuario.
  • Generación de salida adecuada: Luego, debes generar la salida deseada. En este ejercicio, la salida esperada es saludar al usuario por su nombre. Debes utilizar la variable que contiene el nombre del usuario para crear la cadena de saludo.

Ejemplo resuelto

C# JS 
                                
    private static void Main(string[] args)
    {
        Console.Write("Ingrese su nombre:");
        string nombre = Console.ReadLine();
        Console.WriteLine("Hola, " +  nombre);
    }
                                
                                
                            
                                
    let nombre = prompt("Ingrese su nombre:");

    alert("Hola, " + nombre);
