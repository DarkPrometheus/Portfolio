Instrucciones

Escriba un programa que reciba como entrada el radio de un círculo y entregue como salida su perímetro y su área:

Resultado esperado

Ingrese el radio: 5 Perimetro: 31.4 Área: 78.5

Explicacion

Este ejercicio es un programa que calcula el perímetro y el área de un círculo basándose en el radio proporcionado por el usuario.

Pasos clave

  • Solicitud de entrada: En este ejercicio, el primer paso es solicitar al usuario que ingrese el radio del círculo. Debes asegurarte de que el usuario proporcione un valor numérico para el radio.
  • Cálculo del perímetro: El siguiente paso implica calcular el perímetro del círculo. El perímetro de un círculo se calcula mediante la fórmula: 2 * π * radio.
  • Cálculo del área: Luego, debes calcular el área del círculo. El área de un círculo se calcula mediante la fórmula: π * radio^2.
  • Generación de salida adecuada: Finalmente, debes generar la salida deseada. En este ejercicio, la salida esperada incluye el perímetro y el área del círculo.

Ejemplo resuelto

C# JS 
                                
    Console.Write("Ingresa el radio: ");
    string input = Console.ReadLine();
    double radio = 0;
    bool done = false;

    while (!done)
    {
        Console.Clear();
        if (double.TryParse(input, out radio))
        {
            done = true;
            Console.WriteLine("Radio: " + radio);
        }
        else
        {
            Console.Write("Ingresa un numero: ");
            input = Console.ReadLine();
        }
    }

    double area = Math.PI * Math.Pow(radio, 2);
    double perimetro = 2 * Math.PI * radio;

    Console.WriteLine("Perimetro: " + perimetro);
    Console.WriteLine("Area: " + area);
